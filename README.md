## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone 
cd 
```
### 2. Install the dependencies

```bash
npm install
```
### 3. Run teh development server

```bash
npm run dev
```

---

## 🌿 Branching Strategy

This project follows a structured Git workflow:

- `main` → stable production-ready code  
- `dev` → active development branch  
- `feature/*` → new features  
- `fix/*` → bug fixes  

All changes are made through pull requests to ensure code quality and consistency.
