# Personal Portfolio

A modern, responsive portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

This project showcases:
- Professional profile and hero section
- Skills, certifications, projects, and experience
- Translation and localization work highlights
- Secure contact form flow with Supabase + Cloudflare Turnstile

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Supabase Edge Functions

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/Ashenafi-Bancha/Ashenafi_portfolio.git
cd Ashenafi_portfolio
```

2. Install dependencies

```bash
npm install
```

3. Run development server

```bash
npm run dev
```

4. Build for production

```bash
npm run build
```

## Available Scripts

- `npm run dev` starts local development server
- `npm run lint` runs ESLint
- `npm run typecheck` runs TypeScript type-checking
- `npm run build` creates production build
- `npm run preview` previews production build locally

## Branching

- `main` stable branch
- `dev` active development branch
